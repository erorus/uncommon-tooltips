Uncommon Tooltips

Background

So, you run a World of Warcraft fansite. You probably link to Wowhead for items and battle pets. And you probably use Wowhead's Tooltips.

However, maybe you're not happy with Wowhead's tooltips. You don't want to rely on ZAM to keep their site performant, and to keep the data accurate. Maybe you don't like the ads on Wowhead. Or, you might prefer to use official Blizzard data sources. For whatever reason, you want to try a new tooltip library. Maybe one you can host yourself.

Another Option

Uncommon Tooltips was built as an alternative World of Warcraft tooltip library for the Web. It gets data directly from Blizzard's Battle.net API, so it's performant, accurate, and up-to-date. The tooltips are designed to match the in-game appearance as much as possible, even using all the official in-game labels direct from the source.

Drop-In Compatibility

If you'd like to try Uncommon Tooltips on your site, but you already use Wowhead tooltips, you won't have to rewrite every link. The Uncommon Tooltips library supports Wowhead's link formats, so all you'd need to do to switch is remove the Wowhead script tag, and insert the Uncommon Tooltips tag.

Installing on Your Site

Since Uncommon Tooltips uses the Battle.net API for data, you'll need to get an API key from them. Fortunately, it's easy and free to get one.

  1. Register for an Battle.net API account if you don't have one yet.
  2. Log in to the Battle.net API Developer Portal.
  3. Click the "Applications" tab.
  4. Click the "Create a New Application" link.
  5. Fill out the form and describe your website.
    • You can mention "Uncommon Tooltips" in the "Describe what your application will do" if you want.
    • You can leave the "Register Callback URL" field blank.
  6. Once your application is registered, find it under the "Keys" or "Applications" tabs.
  7. Copy the "Key" (NOT the "Secret") for your application to your clipboard. You'll use it to initialize the Uncommon Tooltips library.
  8. In your site code, paste this tag on your page:
    <script src="https://js.uncommon-tooltips.com/uncommon-tooltips.js" onload="uncommonTooltips='KEY';" async></script>
  9. Replace the KEY in the above tag with your key you copied in step 7.
  10. Make sure you remove Wowhead's "power.js" script tag if you're already using it.

That sounds like a lot of work.

It's certainly a lot easier to use Wowhead's tooltips on your website. If you're not comfortable with editing HTML and all the steps above, you should probably stick with Wowhead, it's pretty epic. Uncommon Tooltips is offered only as an alternative for those seeking something different.

Self-Hosting Option

We recommend that you link directly to the script, as in the above example, so your site will always get the latest data and functionality updates. However, if you're concerned about performance or privacy, you can host the script yourself.
  1. Download the main script file to save on your site.
  2. Also download the localization json for all the languages you're using:
    English German Spanish French Italian Portuguese Russian Korean
  3. When you include the script, you'll need to tell the script where your locale files are saved. If, for example, all the files are in your /tooltips/ path, it would look like:
    <script src="/tooltips/uncommon-tooltips.js" async></script> <script> uncommonTooltips = {     key: 'KEY',     localesPrefix: '/tooltips/', }; </script>
  4. Icon images are pulled from Blizzard's CDN. If you prefer to use a different icon location, use the same structure as above, but include:
    iconsPrefix: '/icons/',

About the Author

Uncommon Tooltips was created by Erorus, a former Wowhead developer who's also behind The Undermine Journal, Realm Pop, WoW Token Info, and Does the API Work. You can view most of his projects on everynothing.net.

Uncommon Tooltips was made primarily to support The Undermine Journal, but it's also a neat experiment to see how far the Battle.net API can go.

You can check out the source code on Github.
